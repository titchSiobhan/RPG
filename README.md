## Questage

A lightweight roguelike RPG text‑based sim where you create a character, explore the realm, complete quests, fight monsters, and survive a world driven by luck, events, and dynamic encounters.

## Features

- Character create with stats and luck tiers

- Dynamic quests and encounters

- JSON driven world data

- Event system with weighted outcomes

- Combat, exploration and progression

- Lightweight Node/Express backend

## Stack

- Node.js
- Express.js
- TypeScript
- JSON-based data storage

## Setup

**Prerequisites** 
- Node.js
- npm 


## Clone

`git clone <https://github.com/titchSiobhan/rpg>`
`cd backend`

## Install dependencies

npm install


## Start server

`npm run dev`

**Server**

runs on `http://localhost:3000`
